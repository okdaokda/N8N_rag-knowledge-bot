# RAG-бот с базой знаний в Telegram

Telegram-бот, который отвечает на вопросы по загруженным документам. Закидываешь файлы в Google Drive — бот автоматически индексирует их в векторную базу данных и потом отвечает на вопросы, опираясь на эти документы. Поддерживает текстовые и голосовые сообщения.

## Проблема

Компании накапливают базу знаний в документах (регламенты, инструкции, FAQ), но сотрудникам сложно быстро найти нужную информацию. Приходится вручную искать по файлам или спрашивать коллег.

## Решение

Система из двух n8n-воркфлоу:

### Воркфлоу 1 — Загрузка документов (`LOADING`)

1. Следит за папкой в Google Drive (триггер каждый час)
2. При появлении нового файла — скачивает и конвертирует в PDF
3. Извлекает текст из PDF
4. Разбивает на чанки по 800 символов с перекрытием 150 символов
5. Создаёт эмбеддинги через OpenAI
6. Загружает в векторную базу данных Supabase

### Воркфлоу 2 — Бот-ассистент (`INFO`)

1. Принимает сообщение в Telegram (текст или голос)
2. Голосовые сообщения транскрибируются через OpenAI Whisper
3. Ищет релевантные документы в Supabase (векторный поиск)
4. AI-агент (GPT-4o-mini) генерирует ответ на основе найденных документов
5. Сохраняет историю диалога в PostgreSQL (память между сессиями)
6. Отправляет ответ в Telegram

## Стек

| Компонент | Технология |
|-----------|------------|
| Автоматизация | n8n (2 связанных воркфлоу) |
| Интерфейс | Telegram Bot |
| AI-модель | OpenAI GPT-4o-mini |
| Транскрибация голоса | OpenAI Whisper |
| Эмбеддинги | OpenAI Embeddings |
| Векторная БД | Supabase (pgvector) |
| Память диалога | PostgreSQL |
| Хранение документов | Google Drive |
| Обработка текста | Recursive Character Text Splitter |

## Файлы

- `LOADING.json` — воркфлоу загрузки и индексации документов
- `INFO.json` — воркфлоу Telegram-бота с RAG

## Как запустить

1. Импортировать оба `.json` файла в n8n
2. Создать таблицу `documents` в Supabase с поддержкой pgvector
3. Настроить credentials:
   - Google Drive (OAuth2)
   - OpenAI API (для эмбеддингов, GPT-4o-mini и Whisper)
   - Supabase (URL + API key)
   - PostgreSQL (для хранения истории диалогов)
   - Telegram Bot (BotFather token)
4. Указать папку Google Drive для отслеживания в `LOADING.json`
5. Активировать оба воркфлоу
6. Загрузить документы в указанную папку Google Drive — они проиндексируются автоматически
